## 2. AI 句式 vs 人类句式对照

### 2.1 动作描写

| AI 典型写法 | 人类典型写法（来源） |
|-----------|-----------------|
| 他往右前方走了三步——进到 Boss 右手盲区 45 度的角度上。这个站位没有写在任何攻略里——它刻在八百小时《黑暗之魂》的肌肉记忆里。不死院恶魔持刀手同侧肩宽外侧，第一刀就永远碰不到你。 | 韩萧一翻身扯断了所有连接在身上的仪器导线，跌跌撞撞冲向实验室门口。"砰！"撞开房门，外面是一条白色的长走廊。（超神机械师） |
| 他的瞳孔收缩了一下。那把刀从地面拖起，角度偏低，身体重心还在右脚。 | 痛！好痛！头好痛！光怪陆离满是低语的梦境迅速支离破碎，熟睡中的周明瑞只觉脑袋抽痛异常，仿佛被人用棒子狠狠抡了一下。（诡秘之主） |

**关键差异**：AI 动作为了展示策略 → 紧跟解释。人类动作为了推进剧情 → 紧跟后果。

### 2.2 内心独白

| AI 典型写法 | 人类典型写法（来源） |
|-----------|-----------------|
| 他深吸一口气。游戏他不怕。他怕的东西——从光隙里涌进来的灰绿色纹理正在覆盖地砖，一点一点——是这种缝合。 | 该死。该死啊！武小德迈开大步，全力以赴的冲向那个地洞。……——这还跑个毛啊！看来只有死路一条了。（武德充沛） |
| 前摇延长两帧意味着什么——意味着打之前先给你多两帧的预判窗口。对不知道这是什么东西的人来说是狂暴，对知道的人来说是送。 | 没有系统，竟然没有系统！这意味着他几乎没办法改变现状，两天后，他就要戴上镣铐和枷锁，被送往边陲。……"我必须想办法自救，我不能就这样狗带。"（大奉打更人） |

**关键差异**：AI 内心独白 = 分析局势。人类内心独白 = 情绪反应 + 自我说服 + 脏话/俚语。

### 2.3 对话节奏

| AI 典型写法 | 人类典型写法（来源） |
|-----------|-----------------|
| A说→B说→A说→B说，严格交替，每句完整回应上一句 | "系统？"沉默了片刻。"系统……系统爸爸，你出来啊。"依旧寂静无声。"没有系统，竟然没有系统！"（大奉打更人） |
| "XX说道""XX淡淡道"贯穿全篇 | "废物！滚出蓝家！"——这是他们对我说的最后一句话。（黄金三章范例） |

**关键差异**：AI 对话像辩论，每轮完整交替。人类对话有沉默、抢话、自言自语、答非所问。

### 2.4 章末处理

| AI 典型写法 | 人类典型写法（来源） |
|-----------|-----------------|
| 系统通知弹了出来。副本公开结算……他深吸一口气，然后走了进去。 | 就着轻纱般的绯红月光，他侧过脑袋，查看额角的情况。清晰倒映的镜子如实呈现，一个狰狞的伤口盘踞在他的太阳穴位置，边缘是烧灼的痕迹，周围沾满了血污，而内里有灰白色的脑浆在缓缓蠕动。（诡秘之主） |
| 走廊里的蓝光映在消防门的光标上，把几个人的影子拉得变了形。 | 他抓住堂弟的衣袖："你能弄到卷宗吗？税银丢失案的卷宗。"（大奉打更人） |

**关键差异**：AI 章末倾向"安全着陆"——总结+氛围渲染。人类章末 = 抛出具体钩子（一句话行动/一个画面冲击/一个悬念）。

## 3. 人类网文的高置信度共性特征

以下特征在 4 本采样中全部出现，标记为写作时必须遵守：

### 3.1 句式节奏
- **单句段是常态**：至少每 4 段有 1 段只有一句话。作用不是"强调"，而是制造呼吸感。
- **碎片句密集**：紧张处句子变短、变碎。人物恐惧/愤怒时不写完整的"主语+谓语+宾语"。
- **段落疏密对比显著**：最长段和最短段字数比 ≥ 8:1。

### 3.2 叙事视角
- **严格限制视角**：读者只比主角多知道一点点，甚至完全同步。不出现"他不知道的是……"。
- **信息"漏"而非"砸"**：设定通过角色处境和行为自然带出，不以旁白堆砌。
- **信任读者**：展示动作后不解释含义，展示情绪后不贴标签。

### 3.3 情绪表达
- **生理反应 > 情绪标签**：不写"他感到恐惧"，写"后颈的汗毛一根根立起来"。
- **内心独白碎片化**：角色不跟自己长篇大论分析局势，而是短促的自我质疑、脏话、自嘲。
- **情绪有残留**：愤怒不会因为一个念头就转为平静，中间必须有过渡。

### 3.4 对话
- **口语化显著**：有省略、打断、口癖、脏话、自言自语。
- **潜台词驱动**：角色不会在对话中把真实想法全盘托出。
- **现代思维穿插**：即使是异世界/古代背景，穿越主角的内心独白保留现代口语特征。

### 3.5 幽默感
- **绝境中有黑色幽默**：武德充沛的蘸料段落、大奉打更人的"父母双亡的人都不简单"——在紧张处插入意料之外的轻逸。
- **自我调侃**：主角会吐槽自己的处境，不会全程严肃。

## 4. 游戏/系统流专项特征

基于超神机械师 + 武德充沛 + 系统流 2024-2025 样本的共性：

| 特征 | 说明 |
|------|------|
| 系统面板融入叙事 | 系统提示不单独成段展示，而是被角色的反应/吐槽打断 |
| 游戏术语白话化 | "DPS""仇恨半径"等术语必须有角色自己的理解或吐槽包裹 |
| 数值增长 ≠ 爽点 | 升级的爽感来自角色尝试新能力的行为，不是数字跳动本身 |
| 缝合感是设定不是借口 | 如果世界观本身就是缝合怪，要在角色层面体现对这种缝合的态度（困惑/吐槽/利用），而非叙述者代为解释 |

## 5. 使用方式

context-agent 在 Step 1 生成任务书时：

1. **读取本文件**（由 skill 流程自动加载）
2. **将"共性"栏指标翻译为本章具体约束**，写入任务书第 4 段，例如：
   - "本章单句段占比目标 ≥ 25%，紧张处句子要碎"
   - "本章'不是A是B'不超过 2 次"
   - "对话占比 ≥ 30%，至少 1 处口语打断或自言自语"
   - "至少 1 处角色内心情绪爆发（碎片式，非分析式）"
3. **根据当前章节类型选择对照片段**：动作场景 → 参考 §2.1，对话场景 → 参考 §2.3，章末 → 参考 §2.4
4. **不暴露文件名和"风格目标"等词**，只翻译为自然写作提醒
